## 下载文件ps1.sh
	ps1.sh脚本的作用是将PS1变量信息写入用户配置文件，来改变终端命令提示信息的颜色。
	如果你会使用vi编辑器，可不使用这个脚本，而直接修改 '~/.bash_profile' 配置文件(MacOS)即可。 

点击下载:[ps1.sh.zip](https://github.com/zimingwz/macos_command/raw/master/03_terminal_set_color_ps1/ps1.sh.zip)
